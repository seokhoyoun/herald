# 에디토리얼 레이아웃 설계 노트

긴 글은 **시선 이동**이 중요합니다. 리듬을 만들기 위해 아래 규칙을
기준으로 레이아웃을 잡습니다.

## 리듬 만들기

1. 소제목 사이에 충분한 여백 확보
2. 한 문단당 3-4줄 유지
3. 강조 문장을 카드나 인용문으로 분리

> 글은 디자인이 아니라 **편집**입니다.
